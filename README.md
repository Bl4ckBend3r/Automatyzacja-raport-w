# Automatyzacja raportów w Pythonie

Projekt demonstruje pełną automatyzację generowania i wysyłki raportów sprzedażowych.

## Funkcjonalności
- Wczytywanie danych z pliku CSV
- Przetwarzanie i agregacja wyników
- Tworzenie wykresów i raportu PDF
- Automatyczna wysyłka raportu e-mailem

## Wymagania
- Python 3.9+
- Biblioteki: pandas, matplotlib, fpdf
